# SuiviCloud Deployment

SuiviCloud uses a static build generated by Node.

- Source of truth: GitHub repository
- Development: local checkout on Windows
- Production: VPS serving static files with nginx
- Contact API: Node service running from `/opt/suivicloud/contact-server.js`

## Local Workflow

```powershell
cd C:\Cline\SuiviCloud
npm run build
npm run check:links
git status
git add <files>
git commit -m "Describe the change"
git push
```

## Production Layout

nginx serves:

```text
/var/www/suivicloud.xyz/current
```

`current` should point to a timestamped release:

```text
/var/www/suivicloud.xyz/releases/YYYYMMDDTHHMMSSZ
```

The contact API is managed separately:

```text
/opt/suivicloud/contact-server.js
```

## Backup Before Production Changes

Run on the VPS before changing production:

```bash
stamp=$(date -u +%Y%m%dT%H%M%SZ)
mkdir -p /var/www/suivicloud.xyz/backups
tar -czf /var/www/suivicloud.xyz/backups/current-$stamp.tar.gz -C /var/www/suivicloud.xyz current releases
tar -czf /var/www/suivicloud.xyz/backups/contact-server-$stamp.tar.gz -C /opt suivicloud
readlink -f /var/www/suivicloud.xyz/current
```

## Manual Deploy From GitHub On VPS

Run on the VPS after the GitHub repository has the desired commit:

```bash
set -euo pipefail

repo=/opt/suivicloud-repo
if [ ! -d "$repo/.git" ]; then
  git clone https://github.com/garyb0/SuiviCloud.git "$repo"
fi

cd "$repo"
git fetch origin
git checkout master
git pull --ff-only origin master
npm run build

stamp=$(date -u +%Y%m%dT%H%M%SZ)
release="/var/www/suivicloud.xyz/releases/$stamp"
mkdir -p "$release"
cp -a dist/. "$release/"

test -f "$release/fr/index.html"
test -f "$release/en/index.html"
test -f "$release/assets/styles.css"
test -f "$release/assets/app.js"

ln -sfn "$release" /var/www/suivicloud.xyz/current
readlink -f /var/www/suivicloud.xyz/current
```

## Rollback

List previous releases:

```bash
ls -1dt /var/www/suivicloud.xyz/releases/*
```

Point `current` back to a known-good release:

```bash
ln -sfn /var/www/suivicloud.xyz/releases/<release-id> /var/www/suivicloud.xyz/current
```

## Do Not Commit

Never commit:

- `.env` files
- API keys
- passwords
- tokens
- generated `dist/`
- `output/`
- `test-results/`
- local archives
